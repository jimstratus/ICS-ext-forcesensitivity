## [1.0.0] - TBD

### Added
- Initial release of Force Sensitivity Detector
- Automatic detection on user registration
- Configurable base probability (default 5%)
- Ratio-based probability adjustment
- Admin manual trigger and override
- Probability modifiers (member, group, global, event)
- Profile field integration
- Badge display on profiles and posts
- Complete audit logging
- Bulk member operations
- Import/Export settings
- Comprehensive documentation

### Technical
- ICS v4.7.20 compatibility
- PHP 7.4+ / 8.0+ support
- MySQL 5.7+ / MariaDB 10.2+ support
